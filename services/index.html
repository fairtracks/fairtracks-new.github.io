The FAIRtracks ecosystem

The FAIRtracks ecosystem contains an interconnected group of services built around the FAIRtracks draft standard. The services are aimed at gathering, curating, validating, integrating, indexing, and analysing the metadata of genomic tracks.

(Please refer to the overview page for illustrations of the interconnections between the services, both internally within the FAIRtracks ecosystem, as well as to external services.)

Core services

Search engine for genome track metadata

Hierarchical browser of the metadata attributes and all values actually present in the database
Categorical search
Graphical builder of SQL-based search queries
Browse results in table or hierarchical form
Export results in JSON or GSuite formats
REST API for integration into downstream tools and scripts
